## ➤ card-element

Group related content and action.

### Properties

| Property    | Attribute   | Description               | Type      | Default |
|-------------|-------------|---------------------------|-----------|---------|
| `hoverable` | `hoverable` | Makes the card hoverable. | `boolean` | false   |
