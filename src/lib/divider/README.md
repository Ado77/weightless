## ➤ Attributes

| Name | Type | Description | Default value |
| ------- | ------- | ------- | ------- |
| `role` | string | Role of the divider. | `'separator'` |


[![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/colored.png)](#css-variables)

## ➤ CSS variables

| Name | Description |
| ------- | ------- |
| `--divider-color` | Color. |
| `--divider-height` | Height. |
