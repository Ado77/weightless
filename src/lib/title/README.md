## ➤ title-element

Indicate the start of a new section.

### Properties

| Property | Attribute | Description         | Type         | Default   |
|----------|-----------|---------------------|--------------|-----------|
| `level`  | `level`   | Level of the title. | `TitleLevel` | 1         |
| `role`   | `role`    | Role of the title.  | `AriaRole`   | "heading" |
